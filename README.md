# wuswap

A constant-product AMM, built from scratch with Foundry. Work in progress — building in public.

The goal isn't another Uniswap clone: it's a V2-style core engineered to 2026 standards
(transient-storage reentrancy guards, typed custom errors, soldeer, an invariant test suite
that actually earns the name) with every security decision documented and tested.

Unaudited. Testnet only. Do not use with real funds.
